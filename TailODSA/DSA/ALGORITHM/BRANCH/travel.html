Traveling Salesman Problem

The Traveling Salesman Problem (TSP) is a classic optimization problem where a salesman must visit a set of cities, returning to the origin city while minimizing the total travel distance.

Constraints:

  • Each city must be visited exactly once.
  • The salesman must return to the starting city.
  • Travel distances between cities are known.
  • Objective is to minimize the total travel distance.
